# TechFlow Solutions - Sistema de Gestão Logística

## 📋 Descrição do Projeto

O **TechFlow Solutions** é um sistema de gestão logística desenvolvido em Python com interface gráfica Tkinter, que implementa um quadro Kanban para gerenciamento de rotas de entrega. O sistema permite o cadastro de operadores, criação de rotas, atribuição de motoristas e acompanhamento do status das entregas.

## 🚀 Funcionalidades

- **Autenticação de Usuários**: Sistema de login e cadastro de operadores
- **Gestão de Rotas**: CRUD completo (Criar, Listar, Atualizar, Deletar) de rotas de entrega
- **Quadro Kanban**: Visualização organizada em colunas:
  - 📋 A Fazer (To Do)
  - 🔄 Em Trânsito (In Progress)
  - ✅ Entregue (Done)
- **Atribuição de Motoristas**: Vincular motoristas às rotas
- **Gerenciamento de Prioridades**: Classificação em Alta, Média e Baixa prioridade
- **Persistência de Dados**: Armazenamento em arquivo JSON

## 🛠️ Tecnologias Utilizadas

- **Python 3.8+**
- **Tkinter**: Interface gráfica
- **Pytest**: Testes automatizados
- **JSON**: Persistência de dados
